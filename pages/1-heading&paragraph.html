HTML

HTML adalah standar markup language untuk Website.

dengan HTML kamu bisa membuat websitemu sendiri.

HTML itu mudah untuk dipelajari - You will enjoy it!

Struktur HTML

< !DOCUMENT html > : merupakan tag yang digunakan untuk menyimpan informasi versi dari HTML yang dibuat.

< html > :tag ini merupakan dasar/ root dari elemen html. Tag < html > merupakan parent tag dari semua element.

< head > : tag head digunakan untuk menyimpan informasi meta data, judul sebuah halaman, halaman CSS dan lain-lain. Tag ini tidak ditampilkan ke browser

< body > : tag body digunakan menampilkan semua data yang meliputi teks, gambar hingga hyperlink. Semua konten yang ada dan ditampilkan pada halaman html akan dimuat di dalam tag ini.

Apa itu elemen HTML ?

elemen html didefinisikan menggunakan tag pembuka, konten, dan tag penutup:

< tagname > konten < tagname />

HTML Headings dan Paragraph

< h1 > judul
